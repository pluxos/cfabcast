.. HOW TO RUN THIS CODE:

Run sbt install script
----------------------

.. code-block:: 
	$ chmod +x sbt_install.sh
	$ ./sbt_install.sh

Run sbt
--------

Please look in the official `documentation of sbt <http://www.scala-sbt.org/release/docs/Getting-Started/Running.html>`_ for more commands.

.. code-block::
    $ ./sbt/bin/sbt
    > compile
    > run
